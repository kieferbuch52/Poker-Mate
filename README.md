# Poker Mate

スマホ向けのポーカー収支・学習PWAです。

## 主な機能
- リング／トーナメントの収支管理
- 累積収支グラフ、リング時給、MTT ROI
- 店舗別成績、チップ残高、通貨・手動換算レート
- 6-max RFIハンドレンジ表
- パワーナンバー表とM値・必要PN計算
- ベット額ごとの必要勝率、MDF、ブラフ損益分岐点
- アウト数から次の1枚・リバーまでのドロー勝率を厳密計算
- バンクロール残高、入出金、リングBI数、MTT参加可能回数
- ハンドメモ
- JSONバックアップ、CSV出力
- オフライン利用

## GitHub Pagesで公開
1. 新しいGitHubリポジトリを作成
2. ZIPを解凍し、このフォルダ内のファイルをすべてリポジトリ直下へアップロード
3. Settings → Pages
4. Sourceを「Deploy from a branch」にする
5. Branchを `main`、Folderを `/ (root)` にして保存
6. 公開URLをChromeで開き、「ホーム画面に追加」

## 注意
- データは端末のブラウザ内（localStorage）に保存されます。
- 機種変更やブラウザデータ削除に備え、定期的にJSONバックアップを保存してください。
- 換算レートは自動更新ではなく手動入力です。
- レンジとパワーナンバーは学習用の目安です。ICM、レーキ、相手傾向、スタック深度に応じて調整してください。
