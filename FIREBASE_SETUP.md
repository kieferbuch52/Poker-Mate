# Poker Mate Googleログイン設定

1. FirebaseでプロジェクトとWebアプリを作成
2. AuthenticationのGoogleプロバイダを有効化
3. Authorized domainsへGitHub Pagesのドメイン（例：username.github.io）を追加
4. Poker Mateの「管理 → 設定 → Googleアカウント → Firebase接続設定」にWeb設定を入力

Googleログインはユーザー識別用です。収支などは端末内保存のままで、自動クラウド同期は行いません。

サービスアカウントJSON、秘密鍵、クライアントシークレットは入力・公開しないでください。
